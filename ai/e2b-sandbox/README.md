# E2b Sandbox

E2B is open sourced on [github](https://github.com/e2b-dev/e2b).

[Self-hosted guidance](https://github.com/e2b-dev/infra/blob/main/self-host.md).

> E2B account is required. Every new E2B account get $100 in credits. You can sign up [here](https://e2b.dev/auth/sign-up).

Get started with [here](https://e2b.dev/docs).

Each sandbox is:

- A small, dedicated, persisted Linux VM
- Fully isolated (hardware-level via [Firecracker](https://github.com/firecracker-microvm/firecracker))
- Ephemeral (destroyed after use)
- **Programmable via [SDK](https://e2b.dev/docs/sdk-reference/js-sdk/v2.14.1/errors)**
