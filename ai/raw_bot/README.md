# vide coding practice

A vide coding practice using coding agent based on the proposal [plan.md](plan.md).

Let LLM polish and generate some specification like [spec.md](spec_en.md).

Then Coding Agent implement accordingly.

# learning summary

1. Proposal should be concise, and clear, consistent.
2. Specification is good, but not silver bullet. Agent might just take it as a reference.
3. Tweaks, adjustments, re-design, re-implementation.
4. This project use [Doubao](https://www.doubao.com/about), it would be much better if using `Claude Code`.
