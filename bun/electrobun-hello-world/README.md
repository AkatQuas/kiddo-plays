# hello-world

A minimal hello-world project built with [`electrobun`](https://blackboard.sh/electrobun/docs/guides/hello-world/).

Mastering [bun](https://bun.com/docs) fundamentals will help streamline development with electrobun.

Known issue:

- [redundant downloading binary](https://github.com/blackboardsh/electrobun/issues/343)

## Install dependencies:

```bash
bun install
```

## Run Development Mode

```bash
bun dev
```
