# Notedly

This is an API server, built on Nest / GraphQL / Typegoose.

The authorization token could be required using RESTful API.

Then you're good to play with the graphql playground.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
