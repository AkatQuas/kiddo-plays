# 生成带授权信息的URL

[原文档](https://support.huaweicloud.com/sdk-python-devg-obs/obs_22_1301.html)

**典型业务落地场景**

- 企业系统：用户上传附件，后端返回 PUT 预签名 URL，前端直传 OBS；下载时动态生成 GET 临时链接预览；
- 文件分享平台：私有文件生成短时分享链接，设置 10 分钟 / 2 小时有效期；
- 大文件分片上传：前端直接 PUT 文件到 OBS，不走业务服务；后端同时支持分段上传全套签名 URL：初始化分片、上传分片、合并分片，适配大文件上传场景。；
- 图片在线处理：用户访问私有图片，附带缩放、裁剪参数的预签名链接；
- 对外合作场景：给合作方临时开放文件上传 / 删除权限，无需创建子账号。

**配套约束与配套能力**

- 跨域场景：前端调用签名 URL 会跨域，需 OBS 控制台配置桶 CORS 规则；
- Header 一致性：上传场景生成签名时传入的 Content-Type，前端请求必须完全相同，否则签名失败；
- 支持永久 AK、临时 STS 密钥两种鉴权模式生成签名 URL。
